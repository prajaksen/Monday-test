Local Setup

1. Start Minikube

minikube start

2. Add the Langfuse Helm repository

helm repo add langfuse https://langfuse.github.io/langfuse-k8s
helm repo update

3. Create a local values file

Create a file named values-local.yaml locally and populate it with the required secrets. Do not commit this file.

4. Deploy Langfuse

helm install langfuse langfuse/langfuse \
  --namespace langfuse \
  --create-namespace \
  -f values-local.yaml

5. Verify pods

kubectl get pods -n langfuse

6. Access the UI

kubectl port-forward svc/langfuse-web 3000:3000

Then open http://localhost:3000 in a browser.

7. Run the webhook locally

python webhook.py

The webhook listens on port 5000 by default.
